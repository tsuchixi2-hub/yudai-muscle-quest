POINT & LEVEL GUIDE

どうすればレベルが上がるか

優大さん版は「重さや回数を伸ばすこと」よりも、部位ごとに休息を入れながら続けることを中心にポイント化します。同じ部位は1日休んで2日目に戻るのをベスト、2日休んで3日目までは許容として見ます。

レベル一覧

ポイントが増えると称号が変わります。重量よりも、部位ごとの休息管理と継続リズムを評価します。

Lv.1 / 0ptリズム開始まずは記録を始める重量よりも、どの部位をいつやったかを残す段階。
Lv.2 / 80pt休息メモ係休ませる意識が出てきた同じ部位を連日やりすぎない流れを作る。
Lv.3 / 180pt週リズム見習い週の中で戻ってこられる空きすぎた部位を見つけて戻す。
Lv.4 / 320pt部位ローテーター上半身と下半身を回せる胸・背中・脚・腹などを偏らせず回す。
Lv.5 / 500pt継続ビルダー1日休息の型が見えてきた同じ部位へベストタイミングで戻れる。
Lv.6 / 720pt休息マスター休む日も計画の一部やりすぎと空きすぎの両方を避ける。
Lv.7 / 980pt7日リズム職人毎日どこかを動かせる部位を変えながら1週間の流れを作る。
Lv.8 / 1280pt習慣アスリート継続が自然になってきた短い日でも記録に戻ってこられる。
Lv.9 / 1620pt部位管理プロ空きすぎを自分で直せる丸3日以上休みっぱなしの部位をすぐ回収できる。
Lv.10 / 2000pt筋肉クエスト王休息と継続の完成形重さよりも、続ける仕組みが体に残る。

狙う順番

最短でレベルを上げるなら、この順番がわかりやすいです。

1. 毎日どこかの部位を記録する追い込まなくてもOK。短い1種目でも、継続の土台として加点。
2. 同じ部位は連続させすぎない胸の翌日は脚や背中など、部位を変えて休息を作る。
3. 1日休んで2日目に同じ部位へ戻るこのモックでは、前回から1日休息を入れて2日目に戻るとベスト加点。
4. 2日休息までは許容、丸3日休みっぱなしは避ける2日休んで3日目に戻るのは許容。丸3日以上同じ部位が空いたら減点。

ポイントの増やし方

モック版の計算ルール。正式版ではここを設定画面から変えられるようにできます。

行動ポイント条件
筋トレ記録を1件保存+6pt部位・種目・重量・回数を入力。記録は小さく加点。
今週の実行日+8pt / 日同じ日に複数種目をやっても、継続日としては1日分。
ベスト部位間隔+18pt同じ部位を1日休ませ、2日目に実行できたとき。
許容部位間隔+10pt疲労が強く、2日休ませて3日目に戻ったとき。
部位バリエーション+10pt / 部位胸だけ、脚だけに偏らず複数部位を回す。
7日リズム特別ボーナス+120pt1週間毎日実行し、前日と同じ部位を連続させない。
同じ部位を連日-8pt休息が短すぎる扱い。やりすぎ防止。
部位が丸3日以上空く-15pt / 部位最後の実行後、3日連続で同じ部位が未実行のとき。
重量・回数アップ加点なし成長記録として表示は残すが、レベル上げでは優先しない。
このモックのルールは医学的な断定ではなく、習慣化しやすくするための仮設です。背景にある考え方は 超回復と筋肥大理論 にまとめています。
